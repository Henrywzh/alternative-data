---
description: Stage all changes, commit, and push to the remote repository
---

This workflow helps you quickly sync your local changes to GitHub.

1.  Stage all current changes in the repository:
// turbo
```bash
git add .
```

2.  Commit the changes with a message:
> [!TIP]
> Make sure to provide a descriptive commit message.
// turbo
```bash
git commit -m "chore: updated by antigravity workflow"
```

3.  Push the committed changes to the remote:
// turbo
```bash
git push
```
