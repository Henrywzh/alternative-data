# alternative-data

Research repo for gathering and analyzing alternative data.

Initial focus areas:

- Web scraping public signals and usage patterns
- OpenRouter API usage exploration
- Additional alternative data projects over time

## Planned structure

- `projects/` for standalone data collection experiments
- `data/` for local datasets or exports that should usually stay out of git
- `notebooks/` for analysis and prototyping
- `src/` for shared utilities

## Notes

This repository is intended as a home for small, practical alternative data projects that can expand over time.
